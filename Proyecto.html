El servidor WebTIS brinda servicio de alojamiento WEB (Web hosting) por parte del Departamento de Informática - Sistemas de la UMSS por medio del Laboratorio de Cómputo, principalmente a estudiantes de la materia de Taller de Ingeniería de Software.



El servicio es temporal, las cuentas creadas tienen una vigencia máxima que es especificada al darse de alta.

ESPECIFICACIONES


Especificaciones técnicas del servidor





BASE DE DATOS

  • MariaDB 10.6.3
  • PostgreSQL 12.7
  • MySQL




BAKEND


  • Laravel 8
  •     Php 7.4.22
  •     Apache 2.4.38L




FRONTEND

  • Bootstrap
  • Angular
  • React
